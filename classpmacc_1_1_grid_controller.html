Detailed Description

template<unsigned DIM>
class pmacc::GridController< DIM >

GridController manages grid information.

GridController provides information for a DIM-dimensional grid such as the number of GPU nodes and the current node's position in the grid and manages sliding window. GridController is a singleton.

Template Parameters
DIMdimension of the controlled grid

◆ slide()

template<unsigned DIM>
bool pmacc::GridController< DIM >::slide ( )
inline

Initialises a slide of the simulation area.

Starts a slide of the simulation area. In the process, GPU nodes are reassigned to new grid positions to enable large simulation areas to be computed. All nodes in the simulation must call this function at the same iteration.

Returns
true if the position of the calling GPU is switched to the end, false otherwise
