template<class T_ValueType, unsigned T_dim, bool T_isSmallValue>
class pmacc::TaskSetValue< T_ValueType, T_dim, T_isSmallValue >

Set all cells of a GridBuffer on the device to a given value.

T_ValueType = data type (e.g. float, float2) T_dim = dimension of the GridBuffer T_isSmallValue = true if T_ValueType can be send via kernel parameter (on cuda T_ValueType must be smaller than 256 byte)
