Detailed Description

template<class TYPE, unsigned DIM>
class pmacc::MappedBufferIntern< TYPE, DIM >

Implementation of the DeviceBuffer interface for cuda mapped memory.

For all pmacc tasks and functions this buffer looks like native device buffer but in real it is stored in host memory.
