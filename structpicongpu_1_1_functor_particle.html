Detailed Description

template<uint32_t r_dir, uint32_t num_pbins, typename SuperCellSize>
struct picongpu::FunctorParticle< r_dir, num_pbins, SuperCellSize >

Functor called for each particle.

Every particle in a frame of particles will end up here. We calculate where in space the owning (super) cell lives and add the particle to the shared memory buffer for that phase space snippet the super cell contributes to.
