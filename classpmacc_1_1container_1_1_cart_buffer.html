Detailed Description

template<typename Type, int T_dim, typename Allocator = allocator::EmptyAllocator, typename Copier = mpl::void_, typename Assigner = bmpl::vector<bmpl::_1, bmpl::_2>>
class pmacc::container::CartBuffer< Type, T_dim, Allocator, Copier, Assigner >

Implementation of a box-shaped (cartesian) container type.

Holds a reference counter so one can have several containers sharing one buffer. Is designed to be an RAII class, but does not fully obey the RAII rules (see copy-ctor). The way memory gets allocated, copied and assigned is fully controlled by three policy classes.

Template Parameters
Typetype of a single value
T_dimdimension of the container
Allocatorallocates and releases memory
Copiercopies one memory buffer to another
Assignerassigns a value to every datum of a memory buffer

Assigner policy has to support apply2: Assigner<Dim, CartBuffer>
