Detailed Description

template<typename T_Slave>
class picongpu::plugins::multi::Master< T_Slave >

Master class to create multi plugins.

Create and handle a plugin as multi plugin. Parameter of a multi plugin can be used multiple times on the command line.

T_Slavetype of the plugin (must inherit from ISlave)
