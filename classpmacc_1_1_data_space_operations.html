Extends the DataSpace object ds of dimension DIM to a DataSpace object of dimension DIM+1.

The direction of extension is passed as an exchange direction in ex.

The extension maps a element with DIM-position ds into a (DIM+1)-position in a (DIM+1)-dimensional grid. The actual grid to map into is constructed as (target)-(offset).

Parameters
dsDataSpace object to extend (e.g. a DIM-position)
exexchange direction for extension
targetDataSpace describing size of target grid
offsetDataSpace describing size of target grid's offset
Returns
extended DataSpace with dimension DIM+1

Reduces the DataSpace object ds of dimension DIM to a DataSpace object of dimension DIM-1.

The direction of reduction is passed as an exchange direction in ex.

The reduction eliminates the first (and only the first) occuring direction in the test order x-y-z.

exexchange direction for reduction
