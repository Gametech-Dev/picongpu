template<typename T_ValueFunctor = pmacc::nvidia::functors::Assign, typename T_ActualPush = void>
struct picongpu::particlePusherProbe::Push< T_ValueFunctor, T_ActualPush >

Probe electro-magnetic fields and store the result with a particle.

Template Parameters
T_ValueFunctorpmacc::nvidia::functors::*, binary functor handling how to store the obtained field on the particle, default is assigning a new value
T_ActualPushallows to perform a real particle push after probing the electro-magnetic field (e.g. to let a probe particle stream with a moving window or to define a tracer particle species that records its fields), default is void and means no push (just a static probe)
