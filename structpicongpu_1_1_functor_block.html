Detailed Description

template<typename Species, typename SuperCellSize, typename float_PS, uint32_t num_pbins, uint32_t r_dir, typename T_Filter, uint32_t T_numWorkers>
struct picongpu::FunctorBlock< Species, SuperCellSize, float_PS, num_pbins, r_dir, T_Filter, T_numWorkers >

Functor to Run For Each SuperCell.

This functor is called for each super cell, preparing a shared memory buffer with a supercell-local (spatial) snippet of the phase space. Afterwards all blocks reduce their data to a combined gpu-local (spatial) snippet of the phase space in global memory.
